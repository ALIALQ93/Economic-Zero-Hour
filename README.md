# ⚡ Economic Zero Hour

> **السيطرة على العالم لا تحتاج جيوشاً — تحتاج رأس مال وذكاءً واستراتيجية.**

لعبة استراتيجية زمن حقيقي (RTS) ثلاثية الأبعاد في المتصفح، مستوحاة من **Command & Conquer: Generals Zero Hour**، يستبدل فيها كل المحتوى العسكري بأدوات اقتصادية. الهدف: السيطرة على دول الخريطة ليس بالجيوش، بل بالمال والنفوذ والتحكم في الموارد.

## الرؤية

### الشخصية المزدوجة للاعب

اللاعب يعيش وجهين في نفس الوقت:

**الواجهة العلنية** — رجل أعمال محترم يوقّع عقوداً، يفتتح مصانع، يُقدّم نفسه كشريك تنمية. العالم يصفق له.

**خلف الكواليس** — هو من يتحكم في أسعار العملة، يموّل المعارضة، يشتري وزراء، ويخنق اقتصادات بأكملها بديون مدروسة.

### الطبقتان البصريتان

```
الطبقة العلنية  →  أرقام وتجارة ونمو وعلاقات دبلوماسية — تبدو نظيفة
الطبقة الخفية   →  خيوط نفوذ حقيقية تظهر فقط للاعب عبر زر Tab
```

### الشعور الذي يحسه اللاعب

| الشعور | التجسيد في اللعبة |
|---|---|
| **التخطيط البارد** | كل قرار له تبعات بعد دقائق — مثل شطرنج على خريطة العالم |
| **التوتر الخفي** | المنافسون يتحركون في الخفاء ولا تعرف متى تنهار دولة آمنة |
| **نشوة السيطرة** | لحظة تدرك أن دولة بأكملها أصبحت تابعة بدون رصاصة واحدة |

## المكدس التقني

```
Frontend   →  TypeScript + Babylon.js (3D في المتصفح)
Physics    →  Babylon.js Built-in Physics Engine
UI Layer   →  Babylon.js GUI + Custom HTML Overlay
State      →  Custom Game State Manager (TypeScript)
AI         →  Custom Behavior Tree System
Audio      →  Babylon.js Audio Engine
Build      →  Vite
Editor     →  Cursor (AI-powered VS Code)
```

### لماذا Babylon.js؟

- مبني خصيصاً للمتصفح بأداء WebGL عالٍ
- TypeScript بشكل أصلي — لا إعداد إضافي
- نظام كاميرا وإضاءة وخريطة جاهز
- أخف بكثير من Unity WebGL (10-20 MB مقابل 150+ MB)
- مجتمع نشط وتوثيق ممتاز

## الفصائل الثلاث

### 🏢 الشركات العابرة للحدود
> *"التمدد الهادئ والاستحواذ التدريجي"*

تدخل الدول كشريك تنموي وتبني مصانع وتفتح فرص عمل، حتى تصبح شرياناً اقتصادياً لا يمكن قطعه. بعدها تبدأ تفرض شروطها.

### 🏦 البنوك والصناديق السيادية
> *"الصبر والفخ"*

تقدم قروضاً بشروط مغرية، ثم تراكم الفوائد حتى تعجز الدولة عن السداد. عندها تطلب "إصلاحات" تخدمها هي.

### ⚡ تكتل الموارد الطبيعية
> *"الخنق البطيء"*

تتحكم في النفط والغذاء والمعادن. الدول المعادية تجد نفسها فجأة بلا طاقة، بلا غذاء، وبلا مواد أولية.

## نظام الموارد

### الموارد الأساسية (تتدفق كل 5 ثوانٍ)

```typescript
interface Resources {
  capital: number;   // رأس المال  — يُجمع من الدول التابعة والاستثمارات
  oil: number;       // النفط      — يُجمع من حقول النفط المسيطر عليها
  food: number;      // الغذاء     — يُجمع من الأراضي الزراعية
  metals: number;    // المعادن    — يُجمع من المناجم المسيطر عليها
}
```

### قانون الشح

| المورد الناقص | ما يتجمد فوراً |
|---|---|
| رأس المال | جميع العمليات الاقتصادية |
| النفط | وحدات النقل والعمليات الثقيلة |
| الغذاء | استقرار الدول التابعة |
| المعادن | الوحدات والمباني التكنولوجية |

### تدفق الموارد على الخريطة

```
خطوط خضراء متحركة  →  موارد تتدفق لصالحك
خطوط حمراء متحركة  →  موارد تتدفق للخصم (يمكن قطعها)
خطوط رمادية        →  تجارة محايدة قابلة للاستحواذ
```

### الموارد الناعمة (غير مرئية لكن حاسمة)

| المورد | كيف تبنيه | تأثير نفاده |
|---|---|---|
| النفوذ السياسي | شراء مسؤولين وتمويل أحزاب | لا يقبلك أحد |
| السمعة الدولية | الاتفاقيات والمساعدات | تُرفض عقودك |
| الثقة الاقتصادية | الوفاء بالاتفاقيات | تُغلق الأسواق |
| الغموض | العمليات الخفية | الخصم يعرف خططك |

## ميكانيكا اللعب

### دورة اللعب — 3 مراحل

```
0-10 دق    →  التأسيس   : بناء المقر + أولى الوحدات + السباق على الدول الاستراتيجية
10-35 دق   →  التمدد    : شراء النفوذ + بناء الموارد + مراقبة المنافسين
35-45 دق   →  الحسم     : أسلحة ثقيلة + Super Weapon + الانهيارات المتسلسلة
```

### نظام التحكم

**التحكم المباشر:**
```
ضغط على وحدة        →  تحديدها
ضغط على دولة        →  إرسال الوحدة إليها
ضغط مزدوج على دولة →  أمر طارئ فوري
سحب مجموعة         →  تحديد عدة وحدات
```

**القائمة الدائرية (ضغط على دولة):**
```
        [ استثمر ]
[ تجسس ]           [ أقرض ]
        [ الدولة ]
[ هاجم ]           [ تحالف ]
        [ اشترِ ]
```

### تفاعل الموارد

```
النفط + الممرات البحرية    =  تحكم كامل في أسعار الطاقة
رأس المال + النفوذ         =  شراء أي قرار سياسي
التكنولوجيا + البيانات    =  كشف خطط الخصم مسبقاً
الغذاء + الممرات          =  تجويع دولة بدون رصاصة
المعادن + التكنولوجيا     =  احتكار صناعة كاملة
```

## نظام النفوذ

كل دولة لها مقياس نفوذ من 0 إلى 100%:

```typescript
enum InfluenceLevel {
  OUTSIDER   = '0-25%',    // دخيل — مقاومة شعبية
  PARTNER    = '25-50%',   // شريك — عقود كبيرة مفتوحة
  DOMINANT   = '50-75%',   // مهيمن — تتحكم في الحكومة
  CONTROLLED = '75-100%',  // سيطرة كاملة — الدولة تابعة
}
```

### الإجراءات وتأثيرها

| الإجراء | التأثير |
|---|---|
| فتح مكتب استثمار | +5% |
| منح قرض | +8% |
| بناء مصنع محلي | +12% |
| شراء مسؤول | +15% |
| مساعدة في أزمة | +10% |
| فضيحة مكشوفة | -20% |
| هجوم اقتصادي مكشوف | -30% |

## نظام الفوز والخسارة

### مستويات الفوز

**فوز جزئي** → السيطرة على منطقة كاملة بنفوذ يتجاوز 75%

**فوز استراتيجي** → قطع 3 موارد أساسية عن الخصم في وقت واحد

**فوز مطلق** → أحد المسارات الأربعة:

| المسار | الشرط | الجنرال المناسب |
|---|---|---|
| الاحتكار العالمي | السيطرة على 5 من 6 مناطق | المستثمر الذكي |
| الانهيار الكبير | تدمير اقتصادات جميع الخصوم | محرك الأسواق |
| الحصار الكامل | قطع النفط والغذاء والممرات معاً | أمير النفط |
| الشراء الكامل | شراء حكومات جميع الخصوم | الظل السيادي |

### أسباب الخسارة

```
الإفلاس          →  رأس المال = 0 ولا موارد تعوضه
الثورة الداخلية  →  جميع الدول التابعة تتمرد دفعة واحدة
العزلة الدولية   →  السمعة = 0 ولا أحد يقبل عقودك
الكشف           →  خططك انكشفت وتحالف الخصوم ضدك
```

### التقرير النهائي الأخلاقي

بعد كل مباراة — حتى لو فزت:

```
الدول التي دمر اقتصادها      →  كم مليون متضرر؟
الموارد التي احتكرتها        →  رفعت أسعارها كم بالمئة؟
قراراتك الأخلاقية           →  كم مرة اخترت الطريق الأصعب؟
تصنيفك النهائي              →  من المحرر الاقتصادي إلى المستغل المطلق
```

npm >= 9

```bash
git clone https://github.com/username/economic-zero-hour.git
cd economic-zero-hour
npm install
```
